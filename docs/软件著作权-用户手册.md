## 1. 软件概述

### 1.1 软件简介

HSVJEngine是一款专业的音视频表演系统，专为Android 11/12平台设计。该系统提供实时音视频处理、多图层渲染、音频效果联动、网络控制等功能，适用于KTV、演出、展览等专业场景。

### 1.2 主要功能

- **高性能渲染**：基于Vulkan 1.1+图形API，支持硬件加速
- **多图层系统**：支持视频、图片、文本、二维码、特效等多种图层类型
- **音频效果联动**：实时音频分析，驱动视觉效果
- **多协议网络控制**：HTTP、TCP、UDP、WebSocket、RS232/485
- **硬件解码**：支持RKMPP和MediaCodec硬件加速
- **播放列表管理**：支持视频、图片播放列表
- **场景管理**：支持场景切换和配置保存

### 1.3 应用场景

- KTV点歌系统
- 舞台演出视觉效果
- 展览展示系统
- 多媒体互动装置

## 2. 系统访问

### 2.1 访问方式

在浏览器中访问 `http://{设备IP}:8080` 进入Web控制台

### 2.2 系统界面布局

- **顶部系统栏**：显示系统状态、分辨率、CPU/内存使用率、重启按钮
- **左侧导航菜单**：快速切换各功能模块
- **主内容区**：当前模块的功能操作界面
- **底部命令日志**：显示系统执行的命令记录

### 2.3 端口说明

| 端口 | 服务 | 说明 |
|------|------|------|
| 8080 | HTTP主服务 | Web控制台、API接口 |
| 8081 | 移动端服务 | 移动端专用入口 |
| 8089 | VOD点播服务 | 点歌Web界面 |
| 8090 | WebSocket | 实时控制、状态推送 |
| 9000 | TCP命令 | 二进制命令接口 |
| 8000 | UDP命令 | UDP命令接口 |
| 18080 | 设备发现 | UDP广播发现 |

## 4. 图层矩阵

### 4.1 功能说明

图层矩阵是系统的核心功能，用于管理和配置所有图层。

### 4.2 画布配置

分辨率、幕布布局、旋转角度（均为只读）

### 4.3 图层选择

点击左侧图层列表选择要操作的图层。

### 4.4 图层属性配置

#### 4.4.1 位置与尺寸
- **X坐标**：图层在画布上的水平位置
- **Y坐标**：图层在画布上的垂直位置
- **宽度**：图层宽度
- **高度**：图层高度

#### 4.4.2 变换属性
- **旋转**：图层旋转角度（0-360°）
- **缩放X**：水平缩放比例
- **缩放Y**：垂直缩放比例
- **透明度**：图层不透明度（0-1）

#### 4.4.3 图层控制
- **可见性**：控制图层是否显示
- **优先级**：图层堆叠顺序，数字越大越上层

### 4.5 图层类型配置

#### 4.5.1 视频图层
- **加载视频**：选择本地视频文件
- **选择V4L2设备**：选择摄像头或HDMI采集设备
- **播放控制**：播放/暂停/停止/进度跳转
- **音量控制**：调节音量
- **播放速度**：调节播放速率（0.5x-2.0x）
- **循环模式**：循环全部/单曲循环/一次播放

#### 4.5.2 图片图层
- **加载图片**：选择本地图片文件
- **播放列表**：创建图片轮播列表
- **缩放模式**：拉伸/适应/填充/原始尺寸
- **显示时长**：每张图片显示时间
- **淡入/淡出**：切换动画时长

#### 4.5.3 文本图层
- **文本内容**：输入要显示的文字
- **字体选择**：选择字体
- **字号**：设置字体大小
- **颜色**：设置文字颜色
- **描边**：文字描边效果
- **对齐方式**：左对齐/居中/右对齐
- **滚动模式**：启用跑马灯效果
- **滚动方向**：左/右/上/下
- **滚动速度**：调节滚动速度

#### 4.5.4 二维码图层
- **文本内容**：输入要编码的文本
- **尺寸**：二维码大小
- **前景色**：二维码颜色
- **背景色**：背景颜色

### 4.6 切片管理
- **创建切片**：将图层分割为多个区域
- **切片配置**：每个切片可独立设置位置、旋转、缩放
- **删除切片**：移除切片

### 4.7 漫游动画
- **漫游模式**：关闭/左右移动/上下移动/圆形移动/自定义路径
- **移动速度**：调节漫游速度
- **移动范围**：设置移动边界

## 5. 视频控制

### 5.1 功能说明

视频控制模块用于管理播放列表和素材文件。

### 5.2 播放列表管理
- **创建播放列表**：点击「新建列表」→ 输入名称 → 选择目标图层 → 选择循环模式 → 点击「创建」
- **删除播放列表**：选择列表 → 点击「删除列表」
- **播放列表设置**：点击「设置」配置循环模式、绑定图层、DMX ID、幻灯片参数
- **播放控制**：播放列表/设为默认/清空列表

### 5.3 素材管理
- **上传素材**：上传单个/多个文件、文件夹、拖拽上传
- **素材分类**：视频/图片/音乐
- **素材操作**：搜索、刷新、重新编码
- **预览开关**：启用/禁用素材预览

### 5.4 音频控制
- **播放控制**：播放/暂停、重播、下一曲、上一曲
- **音量控制**：音量加减、静音、图层音量、系统音量
- **播放设置**：播放速度、Seek跳转、锁定控制
- **音轨与声道**：音轨切换、声道设置（立体声/左/右）
- **DSP音频路由**：DSP系统/输入/麦克/关闭

### 5.5 流视频测试
- **流URL**：输入网络视频流地址
- **测试播放**：点击测试播放，查看日志状态

## 7. 中控配置

### 7.1 外设配置

配置和控制外部设备，支持DMX512、串口、UDP、TCP四种协议，采用统一的配置结构。

#### 7.1.1 通用配置结构
- **参数配置**：设置通信参数（端口/波特率/地址等）
- **触发指令管理**：添加、搜索、清空触发指令
- **配置功能**：为每个触发指令配置对应的系统功能

#### 7.1.2 DMX512控制（专用）
- **端口类型**：RS485/USB/Art-Net/sACN
- **地址码**：起始DMX地址（1-512）
- **通道监视器**：显示通道1-12的实时值（0-255）
- **映射配置**：配置DMX通道与系统功能的映射

#### 7.1.3 串口配置（专用）
- **串口号**：COM1/COM2/USB等
- **波特率**：1200/9600/19200/115200
- **学习模式**：启用后自动记录接收到的串口指令
- **模板管理**：保存/加载/清空墙板模板
- **功能列表**：按图层筛选可用功能

#### 7.1.4 网络配置（UDP/TCP通用）
- **监听地址**：默认0.0.0.0
- **监听端口**：UDP默认8000，TCP默认9000
- **连接模式**：TCP支持服务器/客户端模式（可选）
- **组播地址**：UDP支持加入组播组（可选）

### 7.2 测试与诊断

#### 7.2.1 发送指令
- **当前调试对象**：显示当前选择的外设类型
- **发送指令**：输入十六进制指令并发送

#### 7.2.2 交互日志
- **日志显示**：显示外设通信的实时日志
- **清空**：清空日志记录

## 10. 常见问题

### 10.1 无法访问Web控制台

**问题**：浏览器无法访问 `http://{设备IP}:8080`

**解决方案**：
1. 检查设备网络连接是否正常
2. 确认设备IP地址是否正确
3. 检查防火墙设置
4. 尝试使用其他浏览器

### 10.2 视频无法播放

**问题**：选择视频文件后无法播放

**解决方案**：
1. 检查视频文件格式是否支持
2. 确认视频文件路径是否正确
3. 查看系统日志获取详细错误信息
4. 尝试重新编码视频文件

### 10.4 外设无响应

**问题**：外设连接后系统无响应

**解决方案**：
1. 检查外设连接是否正常
2. 确认通信参数配置正确
3. 查看外设交互日志
4. 尝试发送测试指令
