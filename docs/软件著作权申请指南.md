## 转换为Word/PDF格式

### 方法1：使用Word直接打开（推荐）

1. **Markdown转Word**：
   - 使用Word打开 `.md` 文件
   - Word会自动识别Markdown格式
   - 另存为 `.docx` 格式

2. **源代码文档转Word**：
   - 使用Word打开 `.txt` 文件
   - 设置字体为 Consolas 或 Courier New
   - 设置字号为 9pt 或 10pt
   - 另存为 `.docx` 格式

### 方法2：使用在线转换工具

- **Markdown转Word**：https://www.markdowntoword.com/
- **Markdown转PDF**：https://www.markdowntopdf.com/
- **TXT转Word**：直接用Word打开即可

### 方法3：使用Pandoc（需要安装）

安装Pandoc后，使用以下命令：

```bash
# Markdown转Word
pandoc 软件著作权-设计说明书.md -o 软件著作权-设计说明书.docx
pandoc 软件著作权-用户手册.md -o 软件著作权-用户手册.docx

# Markdown转PDF
pandoc 软件著作权-设计说明书.md -o 软件著作权-设计说明书.pdf
pandoc 软件著作权-用户手册.md -o 软件著作权-用户手册.pdf
```

Pandoc下载地址：https://pandoc.org/installing.html
