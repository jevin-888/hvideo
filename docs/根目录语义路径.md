# 根目录语义路径

**原则**：配置与 API 中**不出现任何路径字段**，仅使用**文件名**；完整路径由逻辑统一计算：**ROOT_PATH + 语义目录 + 文件名**。

---

## 一、根目录 ROOT_PATH

- **含义**：应用数据根目录，由 `PathConfig::ROOT_PATH` 决定。
- **典型值**：`/huoshan/` 或 `/sdcard/huoshan/`（仅此两种）。
- **使用**：所有资源文件路径 = `ROOT_PATH + 语义目录 + 文件名`，不在配置或 API 中保存/传递完整路径。

---

## 二、语义目录（逻辑指定）

| 语义目录 | 用途 | 配置/API 字段 | 示例文件名 |
|----------|------|----------------|------------|
| `Image/` | 普通图片、切片遮罩 | `image_file`、`mask_file` | `image.png`、`mask.png` |
| `ttf/` | 字体 | `font_file` | `lyric.ttf` |
| `QRCode/` | 二维码图、二维码 Logo | 图层 71 固定 `qrcode_{layerId}.png`；`qr_logo_file` | `qrcode_71.png`、`logo.png` |
| `Logo/` | Logo 图层（图层 70） | 固定，不存字段 | 由逻辑指定 |
| `video/` | 视频 | 播放列表 URI，不在此文档 | — |
| `config/` | 配置 | `CONFIG_PATH` 等，不存于图层 | — |
| `Lyrics/` | 歌词文件 | 歌词逻辑，不存于图层 | — |

**规则**：

- 配置与 API 只出现**文件名**（如 `image_file`、`font_file`、`qr_logo_file`、`mask_file`）。
- 运行时路径由代码计算：`ROOT_PATH + 上表语义目录 + 文件名`。
- **禁止**在 config.json、API 请求/响应中出现 `image_path`、`font_path`、`qr_logo_path`、`mask`（路径形式）等路径字段。

---

## 三、图层相关

| 图层 | 资源 | 字段 | 路径逻辑 |
|------|------|------|----------|
| 图像（60/80 等） | 图片 | `image_file` | `ROOT_PATH + "Image/" + image_file` |
| Logo（70） | Logo 图 | 无（固定） | 由逻辑指定 `ROOT_PATH + "Logo/..."` |
| 二维码（71） | 二维码图 | 无（固定） | `ROOT_PATH + "QRCode/qrcode_71.png"` |
| 文本/歌词 | 字体 | `font_file` | `ROOT_PATH + "ttf/" + font_file` |
| 二维码 Logo | Logo 图 | `qr_logo_file` | `ROOT_PATH + "QRCode/" + qr_logo_file` |
| 切片遮罩 | 遮罩图 | `mask_file` | `ROOT_PATH + "Image/" + mask_file` |
