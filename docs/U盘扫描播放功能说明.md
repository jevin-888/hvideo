# U 盘扫描播放功能说明

本文档说明移动端「U 盘」按钮功能：在手机控制页点击 U 盘后，**扫描设备上已挂载的 USB 存储中的媒体文件**，生成播放列表并支持**顺序、随机、列表循环、单曲循环**四种播放模式。

---

## 一、功能概述

| 项目 | 说明 |
|------|------|
| 入口 | 移动端控制页（`/mobile/`）视频页内的 **U盘** 按钮 |
| 动作 | 点击后扫描 USB 设备（如 U 盘）中的视频/图片/音频文件，生成播放列表并开始播放 |
| 播放模式 | 顺序、随机、循环（列表循环）、单个循环（单曲循环） |

---

## 二、使用步骤

1. **插入 U 盘**  
   将 U 盘插入 Hvideo 设备（Android 设备需具备存储权限，能访问 `/storage/`、`/mnt/` 等挂载点）。

2. **打开移动端控制页**  
   手机与设备同一 WiFi 下，浏览器访问 `http://{设备IP}:8081/` 或 `http://{设备IP}:8080/` 进入移动端页面。

3. **点击 U 盘按钮**  
   在视频页找到「U盘」按钮并点击，系统会：
   - 显示「扫描中…」；
   - 调用后端扫描 USB 挂载目录（如 `/storage/`、`/mnt/`）中的媒体文件；
   - 弹出或内联选择**播放模式**（见下）。

4. **选择播放模式**  
   - **顺序**：按文件列表顺序依次播放，播完停止。  
   - **随机**：随机顺序播放。  
   - **循环**：按列表顺序循环播放（列表循环）。  
   - **单个循环**：当前文件单曲循环。

5. **自动生成播放列表并播放**  
   选择模式后，系统会创建名为「U盘_时间戳」的播放列表，将扫描到的媒体文件加入列表，应用所选模式并开始播放。底部「播放/暂停」「音量」等控制与现有逻辑一致。

---

## 三、播放模式与后端参数对应

| 用户选择 | 后端含义 | API 参数示例 |
|----------|----------|----------------|
| 顺序 | 顺序播放，不循环列表 | `mode: "sequence"`, `loop: 3` |
| 随机 | 随机顺序 | `mode: "random"`, `shuffle: true` |
| 循环 | 列表循环 | `mode: "sequence"`, `loop: 0` |
| 单个循环 | 单曲循环 | `mode: "sequence"`, `loop: 2` |

---

## 四、支持的媒体类型与路径

- **扫描范围**：后端会扫描设备上常见 USB 挂载目录（如 `/storage/`、`/mnt/`）下的媒体文件。  
- **文件类型**（与当前页「视频/图片/音频」Tab 一致）：  
  - 视频：`.mp4`, `.mkv`, `.avi`, `.mov`, `.wmv`, `.flv`, `.webm`, `.m4v`  
  - 图片：`.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`, `.webp`, `.svg`  
  - 音频：`.mp3`, `.wav`, `.flac`, `.aac`, `.ogg`, `.m4a`, `.wma`  
- **路径权限**：播放列表中使用 U 盘绝对路径（如 `/storage/XXXX-XXXX/视频/片头.mp4`），需设备端已允许应用访问该路径（见 [U盘播放与手机控制使用说明](./U盘播放与手机控制使用说明.md)）。

---

## 五、异常与提示

- **未检测到 U 盘媒体**：若扫描结果为空（未插 U 盘、无媒体文件或无权限），页面会提示「未检测到 U 盘媒体」或「请插入 U 盘后重试」，不会创建空播放列表。  
- **扫描超时**：U 盘内文件过多时，扫描可能较慢或超时，可稍后重试或减少 U 盘内文件数量。  
- **拔出 U 盘**：播放列表中的路径为 U 盘绝对路径，拔出 U 盘后当前项会播放失败，需重新插入 U 盘或切换其他来源。

---

## 六、相关 API（供集成参考）

| 功能 | 方法 | 路径 |
|------|------|------|
| 扫描 USB 媒体 | GET | `/api/filesystem/list?path=...` 或 `/api/filesystem/usb/media`（以实际实现为准） |
| 创建播放列表 | POST | `/api/playlist/create` |
| 添加播放项 | POST | `/api/playlist/{id}/add`（body: `path`, `title`） |
| 设置播放模式 | POST | `/api/playlist/{id}/config`（body: `mode`, `shuffle`, `loop`） |
| 播放播放列表 | POST | `/api/playlist/play`（body: `playlist_id`, `layer_id`, `index`） |
