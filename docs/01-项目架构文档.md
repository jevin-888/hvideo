## 1. 项目概述

### 1.1 项目简介

HSVJEngine（火山VJ引擎）是一个专业的音视频表演系统，专为Android 11/12平台设计。该系统提供实时音视频处理、多图层渲染、音频效果联动、网络控制等功能，适用于KTV、演出、展览等场景。

### 1.2 主要特性

- **高性能渲染**：基于Vulkan 1.1+图形API，支持硬件加速
- **多图层系统**：支持视频、图片、文本、二维码、特效等多种图层类型
- **音频效果联动**：实时音频分析，驱动视觉效果
- **多协议网络控制**：HTTP、TCP、UDP、WebSocket、RS232/485
- **硬件解码**：支持RKMPP和MediaCodec硬件加速
- **区域矩阵**：支持输入/输出矩阵映射、区域拼接和多屏布局
- **播放列表管理**：支持视频、图片播放列表
- **场景管理**：支持场景切换和配置保存

### 1.3 应用场景

- KTV点歌系统
- 舞台演出视觉效果
- 展览展示系统
- 多媒体互动装置

## 2. 核心技术栈

### 2.1 图形渲染

- **Vulkan 1.1+**：现代图形API，提供高性能GPU加速
- **着色器**：SPIR-V格式，支持自定义特效
- **DRM/KMS**：直接渲染模式，无需窗口系统

### 2.2 视频解码

- **FFmpeg 8.0.1**：音视频编解码库
- **RKMPP**：瑞芯微硬件解码（RK3588等芯片）
- **MediaCodec**：Android标准硬件解码
- **零拷贝**：DMA-BUF和AHardwareBuffer优化

### 2.3 音频处理

- **AAudio**：Android低延迟音频API
- **Superpowered**：高性能音频处理库
- **FFT分析**：实时频谱分析

### 2.4 数据库

- **SQLCipher**：加密SQLite数据库
- **SQLite**：轻量级关系数据库

### 2.5 网络通信

- **HTTP Server**：基于自研HTTP服务器
- **WebSocket**：实时双向通信
- **TCP/UDP**：原始套接字通信
- **RS232/RS485**：串口通信

### 2.6 开发工具

- **CMake**：跨平台构建系统
- **Android NDK**：原生开发工具包
- **Gradle**：Android构建工具

### 3.2 架构层次说明

#### 3.2.1 应用层
- Java代码，提供Android Activity和UI
- 通过JNI调用C++核心引擎

#### 3.2.2 核心引擎层
- **Engine**：系统核心，管理所有模块生命周期
- **CommandRouter**：命令路由，处理网络控制命令
- **SceneManager**：场景管理，保存和加载场景配置
- **LicenseManager**：授权管理，验证软件授权

#### 3.2.3 功能模块层
- **Layer**：图层管理（视频、图片、文本等）
- **Decoder**：视频解码
- **Audio**：音频处理和播放
- **Effect**：音频效果分析
- **Network**：网络通信
- **Database**：数据库操作

#### 3.2.4 渲染层
- **VulkanRenderer**：主渲染器
- **CaptureRenderer**：采集渲染
- **TextRenderer**：文本渲染

#### 3.2.5 硬件抽象层
- 封装底层硬件API
- 提供统一接口

## 4. 核心模块

### 4.1 模块概览

HSVJEngine包含13个核心模块，每个模块负责特定功能：

| 模块 | 目录 | 主要功能 | 关键类 |
|------|------|----------|--------|
| Core | `src/core/` | 系统核心、命令路由 | Engine, CommandRouter |
| Layer | `src/layer/` | 图层管理 | Layer, LayerVideo, LayerImage |
| Decoder | `src/decoder/` | 视频解码 | VideoDecoder, DecoderCore |
| Renderer | `src/renderer/` | Vulkan渲染 | VulkanRenderer, CaptureRenderer |
| Audio | `src/audio/` | 音频处理 | AudioProcessor, AudioPlayer |
| Effect | `src/effect/` | 音频效果 | EffectManager, WaveBeatDetector |
| Network | `src/network/` | 网络通信 | NetworkManager, HttpServer |
| Database | `src/database/` | 数据库操作 | VodDatabase, PlaylistManager |
| Text | `src/text/` | 文本渲染 | VulkanTextOverlayBridge, MessageHintRenderer |
| Lyric | `src/lyric/` | 歌词渲染 | LyricRenderer, ASSRenderer |
| Capture | `src/capture/` | 视频采集 | V4L2Capture, RgaConverter |
| Renderer | `src/renderer/` | 区域矩阵输出 | RegionRotationRenderer |
| Utils | `src/utils/` | 工具函数 | Logger, FileUtils, JsonUtils |

**渲染与 Vulkan 约定（维护前必读）**：`docs/渲染流程-完整分析.md`、`docs/Vulkan画布与命令缓冲约定.md`、`docs/Vulkan设备丢失排查.md`。

### 4.2 模块详细说明

#### 4.2.1 Core（核心模块）

**职责**：
- 系统初始化和生命周期管理
- 命令路由和处理
- 场景管理
- 授权验证

**关键类**：
- `Engine`：引擎主类，管理所有模块
- `CommandRouter`：命令路由器，处理11种命令类型
- `SceneManager`：场景管理器
- `LicenseManager`：授权管理器
- `SystemConfig`：系统配置
- `PathConfig`：路径配置

**依赖**：所有其他模块

#### 4.2.2 Layer（图层模块）

**职责**：
- 管理多种类型的图层
- 图层渲染和更新
- 图层属性控制（位置、大小、旋转、透明度等）

**图层类型**：
- `LayerVideo`：视频图层
- `LayerImage`：图片图层
- `LayerText`：文本图层
- `LayerQRCode`：二维码图层（继承自LayerImage）
- `LayerEffect`：特效图层

**关键特性**：
- 切片渲染（Slice）
- 漫游动画（Roam）
- 特效支持
- 混合模式

#### 4.2.3 Decoder（解码模块）

**职责**：
- 视频文件解码
- V4L2设备采集
- 硬件加速解码
- 帧同步管理

**关键类**：
- `VideoDecoder`：解码器主接口
- `DecoderCore`：解码核心
- `HardwareManager`：硬件管理
- `FramePool`：帧池
- `FrameSyncManager`：帧同步

**支持格式**：
- 视频：H.264, H.265, VP9等
- 音频：AAC, MP3, PCM等

#### 4.2.4 Renderer（渲染模块）

**职责**：
- Vulkan图形渲染
- 多图层合成
- 特效渲染
- 区域矩阵

**关键类**：
- `VulkanRenderer`：主渲染器
- `CaptureRenderer`：采集渲染
- `RegionRotationRenderer`：区域矩阵渲染

**渲染管线**：
1. 图层渲染到Canvas
2. 特效处理
3. 区域分割
4. 区域矩阵输出
5. 输出到屏幕


#### 4.2.5 Audio（音频模块）

**职责**：
- 音频播放
- 采集音频接收（Android Java push）
- FFT频谱分析
- 音频效果处理

**关键类**：
- `AudioProcessor`：音频处理器
- `AudioPlayer`：音频播放器

**音频流程**：
1. 从视频解码器获取PCM数据
2. FFT频谱分析
3. 传递给EffectManager
4. 通过AAudio播放

#### 4.2.6 Effect（效果模块）

**职责**：
- 音频强度分析
- 节拍检测（BPM）
- 密集鼓点检测
- 高潮检测
- 效果参数计算

**关键类**：
- `EffectManager`：效果管理器
- `WaveBeatDetector`：节拍检测器
- `DiscogsDetector`：音乐风格检测

**检测算法**：
- BPM检测：峰值形态验证 + 间隔聚类
- 密集检测：SuperFlux Onset Detection
- 高潮检测：多特征融合（能量、熵、节拍密度、低音占比）

#### 4.2.7 Network（网络模块）

**职责**：
- HTTP服务器（8080、8081、8089端口）
- WebSocket实时通信
- TCP/UDP通信
- RS232/RS485串口通信
- 设备发现服务

**关键类**：
- `NetworkManager`：网络管理器
- `HttpServer`：HTTP服务器
- `WebSocketServer`：WebSocket服务器
- `TcpServer`、`UdpServer`：TCP/UDP服务器
- `Rs232Server`、`Rs485Server`：串口服务器

**API端点**：
- `/api/system`：系统控制
- `/api/layer`：图层管理
- `/api/video`：视频播放
- `/api/playlist`：播放列表
- `/api/scene`：场景管理

#### 4.2.8 Database（数据库模块）

**职责**：
- VOD点播数据库管理
- 播放列表管理
- 歌曲信息查询

**关键类**：
- `VodDatabase`：VOD数据库
- `PlaylistManager`：播放列表管理器
- `PlaylistDatabase`：播放列表数据库

**数据表**：
- `songs`：歌曲信息
- `playlists`：播放列表
- `playlist_items`：播放列表项

#### 4.2.9 Text（文本模块）

**职责**：
- 文本渲染
- 字体管理
- 文本特效

**关键类**：
- `VulkanTextOverlayBridge`：基础文本纹理桥接
- `MessageHintRenderer`：消息提示渲染

#### 4.2.10 Lyric（歌词模块）

**职责**：
- 歌词文件解析
- 歌词同步显示
- 歌词特效

**关键类**：
- `LyricRenderer`：歌词渲染器
- `ASSRenderer`：ASS字幕渲染器

**支持格式**：
- ASS/SSA字幕格式
- LRC歌词格式

#### 4.2.11 Capture（采集模块）

**职责**：
- V4L2视频采集
- 格式转换
- HDMI输入采集

**关键类**：
- `V4L2Capture`：V4L2采集器
- `RgaConverter`：RGA格式转换
- `VpssConverter`：VPSS格式转换

#### 4.2.12 Region Matrix（区域矩阵）

**职责**：
- 输入画布区域分割
- 输出矩阵布局
- 区域颜色参数和旋转输出

**关键类**：
- `RegionRotationRenderer`：区域矩阵渲染器

#### 4.2.13 Utils（工具模块）

**职责**：
- 日志系统
- 文件操作
- JSON解析
- 字符串处理

**关键类**：
- `Logger`：日志记录器
- `FileUtils`：文件工具
- `JsonUtils`：JSON工具
- `StringUtils`：字符串工具

## 5. 数据流

### 5.1 视频播放数据流

```
VideoDecoder → DecoderCore → FFmpeg/RKMPP → AVFrame
                                                 ↓
                                            FramePool
                                                 ↓
LayerVideo ← getCurrentFramePtr() ← FrameSyncManager
     ↓
VulkanRenderer → Vulkan Pipeline → GPU → Display
```

### 5.2 音频处理数据流

```
VideoDecoder → AudioCallback → PCM Data
                                   ↓
                            AudioProcessor
                                   ↓
                         FFT Analysis → AudioSpectrum
                                   ↓
                            EffectManager
                         ↙              ↘
                  BPM Detection    Intensity Analysis
                         ↓              ↓
                  LayerEffect    Visual Effects
```

### 5.3 网络控制数据流

```
Client → HTTP/WebSocket/TCP/UDP → NetworkManager
                                        ↓
                                  CommandRouter
                                        ↓
                    ┌───────────────────┼───────────────────┐
                    ↓                   ↓                   ↓
                 Engine              Mubu            SystemConfig
                    ↓                   ↓                   ↓
              各模块操作           图层操作            配置修改
```

### 5.4 渲染管线数据流

```
多个Layer → renderLayersToCanvas() → Canvas Buffer
                                           ↓
                                    Effect Processing
                                           ↓
                                    Region Splitting
                                           ↓
                                  Geometry Correction
                                           ↓
                                    Vulkan Present
                                           ↓
                                       Display
```

### 5.5 场景切换数据流

```
User Command → SceneManager.loadScene()
                     ↓
              Read scene.json
                     ↓
         ┌───────────┼───────────┐
         ↓           ↓           ↓
    Layer Config  System Config  Effect Config
         ↓           ↓           ↓
    Mubu.updateLayer()  SystemConfig.apply()  EffectManager.apply()
         ↓           ↓           ↓
    重新渲染    更新系统参数    更新效果参数
```

## 6. 关键设计模式

### 6.1 单例模式
- `NetworkManager`：全局唯一的网络管理器
- `Logger`：全局日志系统

### 6.2 工厂模式
- `Layer`创建：根据类型创建不同的图层子类

### 6.3 策略模式
- 文本渲染器：不同的渲染策略（ASS、Simple、Hint）
- 解码器：不同的硬件加速策略（RKMPP、MediaCodec）

### 6.4 观察者模式
- 音频回调：AudioProcessor通知EffectManager
- 网络事件：WebSocket连接状态通知

### 6.5 对象池模式
- `FramePool`：视频帧对象池，减少内存分配

### 6.6 命令模式
- `CommandRouter`：将命令请求封装为对象
